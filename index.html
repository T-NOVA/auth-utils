Welcome to Gatekeeper project page.

Gatekeeper is a simple, lightweight authentication & authorization service written completely in GO. Its easy to use APIs allows quick integration in several use-cases.

Features

This lightweight service allows

  • user account registration
  • user authentication
  • token generation and validation
  • service resgitration
  • simple access control
